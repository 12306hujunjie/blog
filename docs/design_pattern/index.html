设计模式学习指南

设计模式是解决软件设计中常见问题的最佳实践总结，分为三大类：

创建型模式

  • 单例模式 (Singleton)
  • 工厂模式 (Factory)
  • 建造者模式 (Builder)
  • 原型模式 (Prototype)

结构型模式

  • 适配器模式 (Adapter)
  • 装饰器模式 (Decorator)
  • 代理模式 (Proxy)
  • 组合模式 (Composite)

行为型模式

  • 观察者模式 (Observer)
  • 策略模式 (Strategy)
  • 迭代器模式 (Iterator)
  • 命令模式 (Command)
